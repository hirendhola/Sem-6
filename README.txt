Data Visualization Practical File
Subject Code: 3160717, 6th Semester, Gujarat Technological University

Instructions:
1. Extract the ZIP file to a local directory.
2. Open `index.html` in a modern web browser (e.g., Chrome, Firefox).
3. Ensure the `data/` folder is in the same directory as the HTML files for experiments requiring data files.
4. For `exp5g_map_json.html`, replace "YOUR_API_KEY" with a valid Google Maps API key to view the map.

Experiments:
- exp1_setup.html: Setup instructions for all tools.
- exp2a to exp2d: Graphical shapes with Canvas and SVG.
- exp3a to exp3i: Charts and tables with HTML5 and JavaScript.
- exp4a to exp4f: Advanced charts with D3.js and Canvas.js.
- exp5a to exp5g: Google Charts and Map visualizations.
- dashboard.html: Interactive dashboard with D3.js, Canvas.js, and Google Charts.
